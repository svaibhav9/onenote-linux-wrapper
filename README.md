# OneNote Linux Wrapper

An unofficial Electron wrapper for Microsoft OneNote Web.

## Features

- Native Linux desktop application
- Internal navigation (no browser redirects)
- Automatic Wacom Pen/Eraser switching
- Dedicated application window
- Custom launcher and icon

## Requirements

- Linux (X11)
- Node.js
- xdotool

## Installation

git clone ...
npm install
npm start

## Wacom Support

The wrapper supports automatic tool switching using the Wacom pen side buttons.

- **Upper side button** → Pen
- **Lower side button** → Eraser
